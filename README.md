üé® Customization
Personal Information
Update the following sections in the HTML:

html
<!-- Change name -->
<h1 class="text-4xl md:text-5xl font-bold mb-4">Hi, I'm <span class="text-yellow-300">Your Name</span></h1>

<!-- Update education details -->
<p class="text-gray-600 mb-2">Your Marks/Total Marks</p>

<!-- Modify skills -->
<span class="bg-blue-100 text-blue-800 px-3 py-1 rounded-full text-sm">Your Skill</span>

<!-- Update contact information -->
<p class="text-gray-600">your.email@example.com</p>
